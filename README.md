# CovidReport

    > Следите за вирусом.
    > С точностью до района.

**CovidReport** - сервис, позволяющий каждому человеку сообщить о состоянии своего здоровья, если он заразился корнавирусом. Также, он позволяет просматривать распространение вируса на карте за разные промежутки времени, а также анализировать опасность посещения конкретного места.
